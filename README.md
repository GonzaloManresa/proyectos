# 🌍 Gestión Digital de Agencia de Viajes
Aplicación empresarial desarrollada con Microsoft Power Platform 
para digitalizar la operación completa de una agencia de viajes.

## 🎯 Problema que resuelve
Las agencias de viajes pequeñas y medianas gestionan clientes, 
viajes y pagos en planillas de Excel o papel, lo que genera 
errores, pérdida de información y falta de visibilidad financiera. 
Esta solución centraliza toda la operación en una app móvil.

## ⚙️ Stack tecnológico
- **Microsoft Power Apps** — Aplicación canvas (móvil y web)
- **Microsoft Dataverse** — Base de datos relacional en la nube
- **Power Automate** — Flujos automáticos de negocio
- **Copilot Studio** — Agentes de IA para asistencia operativa

## 🏗️ Arquitectura — 4 módulos
| Módulo | Descripción |
|---|---|
| Clientes | Registro y gestión de perfiles con notas personalizadas |
| Viajes | Alta de viajes con destino, fechas, vuelo, hotel y estado |
| Pagos | Control de cuotas, saldo pendiente y forma de pago |
| Resumen | Dashboard financiero: facturado, cobrado y pendiente |

## 🤖 Automatizaciones implementadas
- Aviso automático de pago próximo a vencer
- Cálculo automático de saldo pendiente por cliente
- Generación de fecha de aviso al registrar un viaje nuevo

## 🚀 Cómo ejecutar el proyecto
1. Importar la solución `.zip` en un entorno de Power Apps
2. Configurar las conexiones de Dataverse
3. Activar los flujos de Power Automate

## 📌 Aprendizajes clave
- Diseño de modelo de datos relacional en Dataverse
- Automatización de procesos de negocio con Power Automate
- Integración de agentes de IA con Copilot Studio
